Detailed Description

Structure to help parsing the command line.

This structure is used by parse_opt() to store command line arguments as they are being parsed.

Parameters
filenameName of the TSP file to be parsed.
memoryMaximum amount of memory (in MB) the program may use. If the program cannot proceed without requesting more memory, it should be terminated. A meaningful value is between 1 and SIZE_MAX. Notice that if the real system does not have enough memory, the program may be terminated by the OOM Killer.
threadsNumber of threads to use.
timelimitMaximum number of seconds the program may run. If the program does not terminate within this time, it should be terminated anyway. A meaningful value is between 1 and SIZE_MAX.
problemPointer to the instance structure to setup.
shouldplotIf true, use GnuPlot to draw the tour. Default: 1.
