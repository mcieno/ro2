◆ plot_instance()

void plot_instance ( instance *  problem)

Plot a TSP instance.

This function will generate a temporary file in the relative path defined by tspplot_tmpfile. It will then spawn a gnuplot process and draw on it.

Parameters
problemPointer to the instance structure.
Note
In order for this function to work properly, GnuPlot binary should be compiled with –enable-datastrings and available in the system PATH.
